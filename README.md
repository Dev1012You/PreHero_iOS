# PrepHero-iOS

https://prephero.com

An app to make highlights video for athletes. 


### Development

* Install Cocoapods and get all dependencies

```
sudo gem install cocoapods
cd PrepHero
pod update
open PrepHero.wcworkspace
```

### Release

Beta Release (Upload to TestFlight and Crashlytics)

```
fastlane beta
```

Production Release

```
fastlane deploy
```
